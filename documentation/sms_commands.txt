Motor start: Starts the motor. Automatically stops after a set time
Motor stop: Stops the motor
Motor time set x: Sets the time on for motor
Motor time get: Sends a sms with the time on for motor
Battery: Sends a sms the voltage levels of the batteries
Sleep time set x: Sets the sleep time
Sleep time get: Sends a sms with the sleep time
Saldo check: Check saldo. Not implemented.
